*** Watermarking Desktop App ***

Allows a user to add a watermark to an image.
Watermarks and images will need to be created beforehand.
Watermarks will be displayed in the upper-left corner.

Required python packages:
Tkinter
Pillow

In order to use the app, run the main.py in a terminal or IDE.
Follow the directions on screen by:
1. Uploading the image you want to receive a watermark
2. Uploading your watermark

The final image will be displayed at which point you can save it to your computer.
